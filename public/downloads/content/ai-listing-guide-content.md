# Chapter 11: Common Mistakes to Avoid

## Mistake 1: Not Adding Details

**Problem:** Entering minimal information
**Result:** Generic, forgettable listings
**Solution:** Spend 2 extra minutes on specifics

## Mistake 2: Not Reviewing Output

**Problem:** Copy-pasting without reading
**Result:** Potential errors, missed customization
**Solution:** Always review before posting

## Mistake 3: Ignoring Social Media

**Problem:** Only using the MLS copy
**Result:** Missing marketing opportunities
**Solution:** Use all generated content

## Mistake 4: One Size Fits All

**Problem:** Using same style for every property
**Result:** Properties don't stand out
**Solution:** Match style to property type

## Mistake 5: Forgetting to Personalize

**Problem:** AI does everything
**Result:** Missing your unique voice
**Solution:** Add local knowledge and personal touches

## Mistake 6: Overwriting

**Problem:** Adding too many adjectives
**Result:** Reads as desperate or untrustworthy
**Solution:** Trust the process, make targeted edits
